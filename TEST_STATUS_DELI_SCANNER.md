## ⚠️ 可能的错误及处理

### 1. 超时错误
```
错误：Request timed out
```
**原因**: 网络延迟或处理时间过长  
**解决**: 
- 已自动重试（3 次）
- 超时设置为 120 秒
- 如仍失败，考虑压缩图片

### 2. API Key 无效
```
错误：Invalid API key
```
**原因**: API Key 配置错误  
**解决**: 检查 `.env` 文件中的配置

### 3. 图片过大
```
错误：图片过大
```
**原因**: 超过 20MB 限制  
**解决**: 使用优化版脚本压缩图片

### 4. 识别失败
```
错误：未识别到题目
```
**原因**: 图片质量或角度问题  
**解决**: 重新拍摄或调整图片

