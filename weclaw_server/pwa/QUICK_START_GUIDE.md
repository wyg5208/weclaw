# PWA 移动端新功能快速上手指南

## 🎯 新增功能

### 1. 📎 图片/文件上传与 AI 问答

现在您可以上传图片、音频、文档等文件，并与 AI 就文件内容进行对话！

#### 快速开始

**步骤 1: 上传文件**
```
1. 打开 WinClaw PWA 聊天界面
2. 点击输入框左侧的 📎 图标
3. 选择要上传的文件（支持多选）
4. 等待上传完成
```

**步骤 2: 查看预览**
- 图片会显示缩略图
- 文档显示文件名和大小
- 可以删除不需要的附件

**步骤 3: 提问**
```
在输入框中输入您的问题，例如：
- "这张图片里有什么？"
- "请总结这个文档的主要内容"
- "这份报告的数据说明了什么？"
```

**步骤 4: 发送并获取 AI 回答**
```
点击"发送"按钮，AI 会分析文件内容并给出详细解答
```

#### 实用场景

📸 **图片识别**
- 上传产品照片 → "这是什么型号？"
- 上传风景照片 → "这是哪里？"
- 上传图表 → "分析这个趋势"

📄 **文档分析**
- 上传 PDF 报告 → "提取关键数据"
- 上传合同 → "找出重要条款"
- 上传论文 → "总结核心观点"

🎵 **音频处理**
- 上传会议录音 → "整理会议纪要"
- 上传讲座录音 → "总结要点"

### 2. 🎤 语音输入

无需打字，说话即可输入文字！利用 Chrome/Edge 浏览器的原生语音识别功能。

#### 快速开始

**步骤 1: 点击麦克风**
```
点击输入框右侧的 🎤 图标
首次使用需要授予麦克风权限
```

**步骤 2: 开始说话**
```
- 看到麦克风图标变红并开始闪烁 → 正在录音
- 清晰地说出您想输入的内容
- 识别结果实时显示在输入框中
```

**步骤 3: 停止录音**
```
- 再次点击 🎤 图标
- 或等待 60 秒自动停止
```

**步骤 4: 编辑并发送**
```
- 检查识别结果
- 可以手动修改
- 确认无误后点击"发送"
```

#### 使用技巧

✅ **提高识别准确率**
- 在安静环境使用
- 说话清晰、语速适中
- 避免背景噪音
- 距离麦克风适当距离

✅ **长文本输入**
- 分段说话，每段之间停顿一下
- 可以说标点符号（部分浏览器支持）
- 定期发送，避免单次输入过长

✅ **便捷操作**
- 走路时不方便打字？用语音！
- 开车时？用语音！（注意安全）
- 做饭时查菜谱？用语音！

## ⚙️ 设置建议

### 麦克风权限设置

**Chrome/Edge:**
```
1. 点击地址栏右侧的锁形图标
2. 找到"麦克风"选项
3. 切换到"允许"
4. 刷新页面
```

**手机浏览器:**
```
1. 首次点击麦克风时会弹出权限请求
2. 点击"允许"
3. 如误点拒绝，需在系统设置中重新授权
```

### 文件类型限制

系统默认支持以下类型：
- 图片：JPEG, PNG, GIF, WebP
- 音频：WAV, MP3, MPEG, WebM  
- 文档：PDF, TXT

**单文件大小限制**: 50MB

## 🐛 常见问题

### Q: 文件上传失败怎么办？

**A:** 
1. 检查网络是否稳定
2. 确认文件未超过 50MB
3. 确认文件类型在支持列表中
4. 尝试刷新页面重试

### Q: 语音识别不准确？

**A:**
1. 换个安静的环境
2. 说话慢一点、清晰一点
3. 确保麦克风没有被遮挡
4. 检查是否是最新版本的 Chrome/Edge

### Q: 点击麦克风没反应？

**A:**
1. 检查浏览器是否支持 Web Speech API
2. 确认已授予麦克风权限
3. 尝试重启浏览器
4. 如仍不行，改用 Chrome 浏览器

### Q: 可以同时上传多少个文件？

**A:**
- 理论上没有数量限制
- 但建议一次不超过 10 个
- 过多文件会影响上传速度和 AI 响应速度

## 🚀 性能优化建议

1. **图片压缩**: 上传超大图片前可适当压缩
2. **分批上传**: 大量文件建议分批次上传

4. **清理缓存**: 定期清理浏览器缓存提升性能
